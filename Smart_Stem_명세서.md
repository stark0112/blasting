# 벤치발파 설계 웹앱 프로그램 명세서

## 1. 개요
- **프로그램 명칭**: Smart Stem (스마트스템)
- **개발 목적 및 필요성**: 발파 설계 시 저항선, 공간격, 장약량 등을 신속하게 계산하여 현장 적용 효율성 향상
- **주요 타겟**: 건설 현장, 광산, 채석장 등 발파 작업이 필요한 현장 기술자

## 2. 주요 기능

### 입력 모듈
- K값, n값 (시험발파추정식 파라미터)
- 허용진동기준치 (cm/sec)
- 보안물건 거리 (m)
- 발파계수, 공간격비율
- 폭약직경 (자동/선택/직접입력)
- 목적 선택 (비산제어/파쇄도개선/광산채석장)

### 설계 모듈
- 저항선(B), 공간격(S), 전색장(T), 장약장(h), 천공장(H), 계단높이 자동 계산
- 장약량/공(Q), 비장약량(c1) 산출

### 계산 알고리즘
- 허용진동 기준 최대 장약량 역산
- 폭약경에 따른 저항선 계산
- 발파패턴 자동 분류 (미진동~대규모발파, 6단계)

### 시뮬레이션/출력
- 발파 패턴 이미지 시각화
- PDF 리포트 생성 및 다운로드
- 인쇄 기능

## 3. 시스템 아키텍처 (웹앱 특성)
- **Front-end**: HTML5, CSS3, JavaScript (Vanilla JS)
- **Back-end**: 서버리스 (클라이언트 사이드 계산)
- **라이브러리**: jsPDF (PDF 생성), html2canvas (화면 캡처)
- **Database**: 없음 (Stateless, 로컬 처리)
- **배포**: GitHub Pages (https://stark0112.github.io/blasting/pwa/)
- **PWA 지원**: Service Worker 기반 오프라인 캐싱, iOS/Android 홈 화면 설치 가능

## 4. 사용자 인터페이스(UI) 설명
- **메인 화면**: 단일 페이지에 입력폼과 결과 동시 표시
- **설계 변수 입력**: 2열 레이아웃으로 파라미터 입력, +/- 버튼으로 값 조정
- **도움말**: 각 입력 필드에 (?) 아이콘 툴팁 제공
- **결과 리포트 출력**: 좌측 표 + 우측 패턴 이미지 (높이 정렬)
- **PDF 저장 / 인쇄 버튼** 제공

## 5. 독창성 및 차별성
- **경량화**: 별도 설치 없이 웹 브라우저에서 즉시 사용
- **모바일 최적화**: 반응형 UI, PWA로 앱처럼 사용 가능
- **오프라인 지원**: Service Worker로 인터넷 없이도 사용 가능
- **자동 패턴 분류**: 계산 결과에 따라 6가지 발파패턴 자동 선택 및 이미지 표시
- **즉시 PDF 출력**: 현장에서 바로 문서화 가능
